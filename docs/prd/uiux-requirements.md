# UI/UX Requirements

## Pages

1. **Page Login/Register** - Authentification Supabase
2. **Page Liste des livres** - Vue principale avec filtres
3. **Modal/Form Ajout** - Formulaire création de livre
4. **Modal/Form Edition** - Formulaire modification de livre

## Composants UI (Shadcn/ui)

- Button, Input, Select pour les formulaires
- Card pour afficher chaque livre
- Dialog/Sheet pour les modales
- Tabs ou ToggleGroup pour les filtres de statut

## Design Principles

- Mobile-first
- Interface épurée, minimaliste
- Actions principales accessibles en 1-2 clics
