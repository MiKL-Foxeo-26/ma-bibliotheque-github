# Product Scope

## MVP - Minimum Viable Product

Les 5 fonctionnalités du brief - ni plus, ni moins :
1. Voir la liste de mes livres
2. Ajouter un livre (titre, auteur, statut)
3. Modifier un livre
4. Supprimer un livre
5. Filtrer par statut (À lire, En cours, Lu)

**Contrainte temporelle :** Quelques jours pour le déploiement.

## Growth Features (Post-MVP)

Explicitement hors scope initial (à réévaluer après MVP) :
- Recherche textuelle dans la collection
- Tri par auteur/titre
- Statistiques de lecture

## Vision (Future)

Hors scope et non planifié :
- API externes (Google Books, Open Library)
- Images de couvertures
- Partage entre utilisateurs
- Notes et critiques
- Scan ISBN
- Catégories/genres
