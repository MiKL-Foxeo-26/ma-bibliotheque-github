# Project Classification

**Type technique :** Web Application (SPA)
**Domaine :** General (productivité personnelle)
**Complexité :** Low
**Contexte projet :** Greenfield - nouveau projet

**Stack technique :**
- Frontend : React 18 + Vite 5.x
- UI : Shadcn/ui + TweakCN
- Routing : React Router 7.x
- Backend : Supabase (auth + database uniquement)
- Déploiement : Vercel

**Contraintes Supabase :** Pas d'Edge Functions, pas de Storage, pas de Realtime
